# 🎯 GHOST PROTOCOL — Anti-Gravity Sniper Game

> **Note:** This project was built as an experiment to try out **Anti-Gravity** (Gemini AI coding assistant) features. It serves as a demo/playground for AI-assisted game development.

## 🎮 Play Now

**[https://shooting-game-fcg1.onrender.com](https://shooting-game-fcg1.onrender.com)**

## About

A full-stack mobile-first sniper shooting game with:
- 3 difficulty levels (Recruit / Operative / Ghost)
- Anti-gravity mechanics
- Realistic 5-layer sniper rifle sound (Web Audio API)
- Scope zoom with canvas scaling
- Leaderboard system

## Run Locally

```bash
npm install
node server.js
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **Frontend:** HTML5 Canvas, CSS, JavaScript, Web Audio API
- **Backend:** Node.js, Express.js
- **Deployed on:** Render
