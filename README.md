# 🎯 GHOST PROTOCOL — Anti-Gravity Sniper Game

A full-stack mobile-first sniper shooting game with difficulty levels, anti-gravity mechanics, and leaderboards.

## Run Locally

```bash
npm install
node server.js
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy on Railway (Free)

1. Go to [railway.app](https://railway.app) and sign up free
2. Click **New Project → Deploy from GitHub**
3. Push your code to GitHub first
4. Connect repo → Railway auto detects Node.js
5. Click **Deploy** → get your live URL in 2 minutes

## Deploy on Render (Free)

1. Go to [render.com](https://render.com) and sign up free
2. Click **New → Web Service**
3. Connect your GitHub repo
4. **Build Command:** `npm install`
5. **Start Command:** `node server.js`
6. Click **Deploy** → get your live URL

## Tech Stack

- **Frontend:** HTML5 Canvas, CSS, JavaScript, Web Audio API
- **Backend:** Node.js, Express.js
- **No database** — in-memory leaderboard (resets on restart)
